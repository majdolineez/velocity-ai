# ⚡ Velocity AI

Application SaaS permettant aux e-commerçants de générer des descriptions produits optimisées SEO grâce à l'IA.

## 🚀 Problématique
Simplifier la création de contenus marketing pour les boutiques en ligne en utilisant l'IA générative pour gagner du temps et améliorer le taux de conversion.

## 🛠️ Stack Technique (2026)
* **Frontend :** Next.js 16 (App Router), React, TypeScript, Tailwind CSS.
* **Backend :** Server Actions Next.js.
* **Data :** PostgreSQL (Neon), Prisma V7.
* **Services :** Clerk (Auth), API Google Generative AI (Gemini 2.5 Flash).

## ✨ Fonctionnalités clés
* **IA Générative :** Storytelling, accroches et avantages optimisés via Google Gemini.
* **Système de Crédits :** Modèle économique intégré avec décompte temps réel.
* **Authentification :** Sécurisation des accès via Clerk.
* **UI/UX :** Design "Glassmorphism" moderne et 100% responsive.
